What are <meta> elements in html? Where are they used? Provide two examples

The meta elements can be found towards the beginning of the file and are used to describe the page and also give any keywords. An example would be how we specify the set keyboard that will be used by saying <meta charset="UTF—8"/> Another example would be when we describe who the author is. If I were the author, I would say <meta name="author" content="Mason Rich" />

Briefly describe the correct usage of the following HTML5 semantic elements: <header>, <article>, <section>, <footer>. Provide an example for each.

The header is used as a container for important information regarding the website. Like we think of when we hear "header", it appears at the top of the page. It can also be a photo. Article is used in order to appropriately separate content within the website, just as would read in a newspaper article. Sections are to designate different parts within the website. Footer is what will appear at the bottom of the webpage and usually includes information like email addresses addresses, contact information, etc.

  1. <header>Mason's Website</header>
  2. <article>Welcome to Mason's website. We are grateful for the time you spend visiting this wonderful website. Plus, you can find all the answers to the first assignment right here!</article>
  3. <section>This is a section of the website</section>
  4. <footer>Mason Rich — masonjamesrich@gmail.com — 8017102531</footer>

Explain the impact, if any, of casing (upper vs. lower) and white space inside your HTML code. How about the impact of casing and spacing within your file naming conventions?

It is necessary when we name our files, that we do so with the correct casing. If our files do not match letter for letter by their casing, then we will have an error or will open up the wrong file. HTML does not care if we upper or lovercase within our code though. For example, <h3> and <H3> produce the same result.Yet, it is always good to practice being consistent. White space is not read on the web browser through HTML. So any spaces within our code are reduced down to one single space between words even if there are 7 spaces. A nonbreaking space or a preformatted text can be used and it will include the spaces.

What is the difference between http:// used to retrieve a webpage in a browser, and ftp:// used to upload pages to the server?

  • FTP stands for File Transfer Protocol and its purpose is to transfer files from one computer to another computer.
  • HTTP stands for Hypertext Transfer Protocol which helps the world wide web access the HTML/webpage
  • HTTP helps us see the websites
  • FTP is used to access and also transfer files.
